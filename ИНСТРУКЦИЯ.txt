1) Для работы программы на сервере нам потребуется 
- Docker
- Nginx (подойдет любой прокси на ваше усмотрение)
- купленный домен адрес 
- хост

2) Подготовка 
- разместить рабочую дерикторию на хосте 
- зайти в нее 
- в папке project в фале const.py добавить токен бота и id адресс канала. в файле seting.py добавить ip адрес хоста в ALLOWED_HOSTS = []
- docker-compose build  (соберет контейнер)
- docker-compose up -d (запустит контейнер в фоновом режиме на 5002 порту локального сервера)

3) Выполнить настройку webhook на доменное имя (адрес) (webhook работает только на защищеных соединениях. Так что потребуется SSL сертификат)
4) Используя прокси, связать доменное имя (адрес) и 5002 порт на локальном хосте
